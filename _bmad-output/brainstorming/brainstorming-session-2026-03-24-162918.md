## Session Overview

- Ferramentas e automações

### Contexto do Dashboard

O dashboard integra dados de marketing e vendas com 6 categorias principais de métricas:

**1. Principal (KPIs Estratégicos)**
- R$ Receita gerada, R$ Gasto em Ads, ROI
- R$ Margem de contribuição
- R$ Pipeline total, Fat. projetado do Pipe, Receita projetada

**2. Premissas (Taxas de Conversão)**
- % CTR, % Connect rate, % Conversão pág. captura
- % Qualified marketing, % Qualified sales
- % Agendamento, % Show-up, % Fechamentos

**3. Números (Volumes do Funil)**
- # Impressões → # Cliques → # View page únicos
- # Lead → # MQL → # SQL
- # Reuniões agendadas → # Reuniões realizadas → # Vendas

**4. Financeiro (Custos de Aquisição)**
- R$ C.P. Lead, R$ C.P. MQL, R$ C.P. SQL
- R$ C.P. Reunião agendada, R$ C.P. Reunião realizada, R$ C.P. Venda

**5. Deltas (Velocidade do Funil)**
- Tempo médio entre cada etapa de conversão
- Tempo médio do lead até venda (ciclo completo)

### Session Setup

Esta é uma auditoria complexa envolvendo múltiplas fontes de dados, cálculos encadeados e dependências entre métricas. A sessão focará em gerar ideias criativas e práticas para garantir a integridade dos dados exibidos.
